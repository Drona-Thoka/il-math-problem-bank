# il-math-problem-bank
Data set of Illinois Math Competitions as well as AMC and AIME. Serves the Illinois Contest trainer discord bot. This set features: ICTM, NSML, AMC 10/12, AIME, and Chicago ARML Tryouts with their solutions.
